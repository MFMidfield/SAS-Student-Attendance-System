# 🌙 Lunar Project - ระบบจัดการการเข้าเรียน

ระบบจัดการการเข้าเรียนดีไซน์สไตล์ **Neubrutalist** ระดับพรีเมียม พัฒนาด้วย **HTML, CSS (Vanilla), JavaScript** และ **Supabase** โปรเจกต์นี้มีเวิร์กโฟลว์ที่แข็งแกร่งสำหรับการติดตามและตรวจสอบการเข้าเรียนของนักเรียน พร้อมระบบจัดการบทบาท (Roles) และการควบคุมระดับผู้ดูแลระบบ

---

## ✨ ฟีเจอร์หลัก (Features)

### 👨‍🎓 ฝั่งนักเรียน (Student Portal)
- **แดชบอร์ดแบบไดนามิก**: แสดงคำทักทายแบบเรียลไทม์, สถิติการเข้าเรียน และตารางเรียนวันนี้แบบสรุป
- **บันทึกการเข้าเรียน**: อินเทอร์เฟซที่ใช้งานง่ายสำหรับส่งบันทึกการเข้าเรียน พร้อมเลือกวิชา, สถานะ (มาเรียน, ลาป่วย, ลากิจ, กิจกรรม) และระบุหมายเหตุ
- **ดูตารางเรียน**: ตารางเรียนเฉพาะส่วนบุคคลที่กรองตามรหัสห้องเรียน (Class ID) ของนักเรียน
- **ตั้งค่าโปรไฟล์**: จัดการข้อมูลส่วนตัวและดูข้อมูลบัญชี

### 👩‍🏫 ฝั่งผู้ดูแลระบบ/อาจารย์ (Admin/Teacher Portal)
- **แดชบอร์ดขั้นสูง**: เมนูนำทางที่ครอบคลุมทุกโมดูลการจัดการ
- **ระบบรอการตรวจสอบ (Pending Logs)**: ดูบันทึกการเข้าเรียนที่ส่งเข้ามาแบบเรียลไทม์ พร้อมระบบค้นหาและกรองตามห้องเรียน
- **การตรวจสอบ 2 ขั้นตอน (Two-Step Verification)**: 
  - **Verify All (อนุมัติทั้งหมด)**: อนุมัติรายการที่กรองอยู่ทั้งหมดพร้อมกัน พร้อมระบบนับถอยหลังเพื่อความปลอดภัย 5 วินาที
  - **Individual Verify (อนุมัติรายบุคคล)**: อนุมัติหรือไม่อนุมัติทีละรายการ พร้อมระบบนับถอยหลัง 3 วินาที
- **ประวัติการตรวจสอบ (Verified Logs)**: หน้าแสดงรายการที่ผ่านการอนุมัติแล้ว พร้อมรายละเอียดว่า *ใคร* เป็นคนอนุมัติ และอนุมัติ *เมื่อไหร่*
- **จัดการตารางเรียน**: สร้าง, แก้ไข และลบตารางเรียนรายวิชาตามห้องและเวลาต่างๆ
- **จัดการผู้ใช้งาน**: จัดการโปรไฟล์นักเรียนและบุคลากร, แก้ไขบทบาท (Role), และอัปเดตรหัสประจำตัวหรือห้องเรียน

## 🎨 ระบบดีไซน์: Neubrutalist
โปรเจกต์นี้ใช้ความสวยงามแบบ **Neubrutalist**:
- **ขอบที่หนาและชัดเจน (Bold Borders)**: ใช้ขอบสีดำหนา (`2px` หรือ `4px`)
- **เงาที่คมชัด (Hard Shadows)**: เงาที่มีคอนทราสต์สูง ไม่เบลอ (`shadow-[4px_4px_0px_#1E1E1E]`)
- **สีสันสดใส (Vibrant Colors)**: ชุดสีที่คัดสรรมาเป็นพิเศษ ทั้งสีพาสเทลที่มีความอิ่มตัวสูงและสีแม่สีที่โดดเด่น
- **แบบอักษร M PLUS Rounded 1c**: การออกแบบตัวอักษรที่ทันสมัยและเป็นมิตรจาก Google Fonts
- **Micro-Animations**: การเปลี่ยนผ่านที่ราบรื่นและเอฟเฟกต์เมื่อวางเมาส์ (Hover) โดยใช้ CSS Keyframes (`fadeIn`, `scaleIn`, `backdrop-blur`)

## 🛠️ เทคโนโลยีที่ใช้ (Technology Stack)
- **Frontend**: Vanilla HTML5, Vanilla CSS3 (ใช้ Tailwind CSS สำหรับ Utility classes ผ่านสคริปต์บนเบราว์เซอร์)
- **Logic**: Vanilla JavaScript (ES6+) แบบ Modular
- **Backend**: **Supabase** (Auth, PostgREST, Row Level Security)
- **Database**: PostgreSQL พร้อม custom triggers และ functions สำหรับซิงค์ข้อมูลอัตโนมัติระหว่าง `auth.users` และ `public.profiles`

## 🚀 การเริ่มต้นใช้งาน (Getting Started)

### สิ่งที่ต้องมี
- เว็บเบราว์เซอร์ที่ทันสมัย
- โปรเจกต์ [Supabase](https://supabase.com/)

### การติดตั้ง
1. Clone repository:
   ```bash
   git clone https://github.com/your-username/lunar-pj.git
   ```
2. ตั้งค่าการเชื่อมต่อ Supabase:
   สร้างไฟล์ `src/lib/supabaseClient.js` พร้อมใส่ข้อมูลของคุณ:
   ```javascript
   import { createClient } from '@supabase/supabase-js'
   const supabaseUrl = 'YOUR_SUPABASE_URL'
   const supabaseKey = 'YOUR_SUPABASE_ANON_KEY'
   export const supabase = createClient(supabaseUrl, supabaseKey)
   ```
3. รันผ่าน local development server (เช่น Vite หรือ Live Server):
   ```bash
   npm run dev
   ```

## 🛡️ สรุปโครงสร้างฐานข้อมูล (Database Schema)
- `profiles`: เก็บข้อมูลโปรไฟล์ผู้ใช้ (ชื่อ, นามสกุล, รหัสประจำตัว, ห้องเรียน, บทบาท)
- `attendance_logs`: ที่เก็บข้อมูลชั่วคราวสำหรับบันทึกการเข้าเรียนที่รอการตรวจสอบ
- `attendance_verify`: ที่เก็บข้อมูลถาวรสำหรับบันทึกการเข้าเรียนที่ผ่านการอนุมัติแล้ว
- `schedule`: เก็บข้อมูลรายวิชา, ห้องเรียน และเวลาเรียน
